## 🚀 使用方法

### 开发者使用

1. **重新打包**:
   ```bash
   python quick_build.py
   ```

2. **清理构建文件**:
   ```bash
   python build.py clean
   ```

3. **生成发布包**:
   ```bash
   python create_release.py
   ```

### 最终用户使用

1. **获取发布包**: 将 `release/` 目录打包分发给用户

2. **运行程序**:
   - macOS/Linux: 运行 `start_app.sh`
   - Windows: 运行 `start_app.bat`
   - 或直接运行可执行文件

3. **访问系统**: 程序会自动在浏览器中打开，或手动访问 `http://localhost:8000`
